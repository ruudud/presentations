Arrow functions


function* naturalNumbers() {
  let current = 0;
  while (true) {
    yield current++;
  }
}
let seq = naturalNumbers();
assert(seq.next() === 0);
assert(seq.next() === 1);
assert(seq.next() === 2);

Arrow functions - taskjs.org


spawn(function*() {
    var data = yield $.ajax(url);
    $('#result').html(data);
    var status = $('#status').html('Download complete.');
    yield status.fadeIn().promise();
    yield sleep(2000);
    status.fadeOut();
});
Every time the task yields a promise, the scheduler blocks the task until the promise is fulfilled.
