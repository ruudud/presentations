The Document Object Model

A W3C standard which lets you …

  • discover elements
  • change elements
  • add and delete elements

DOM tree

Getting an element


var e = document.getElementById('myElement');

var f = document.getElementsByTagName('div')[0];

var g = document.getElementsByClassName('roundedBox')[0];

Walking the DOM


// Given HTML element `e`…

e.parentNode
e.childNodes[0]

document.firstChild.firstChild.lastChild

Manipulating


document.firstChild.firstChild.nodeValue = 'some text';
document.firstChild.setAttribute('id', 'myBox');

document.firstChild.firstChild.style.color = '#f00';

Adding


var el = document.createElement('hr');
document.getElementById('myBox').appendChild(el);

jQuery / Zepto

Why? Cleaner API, smush out browser differences ++


$.get('http://updates.html5rocks.com')
  .success(function (data) {
    $('#html5rocks').html(data);
  })
  .error(function () {
    console.error('Could not connect to html5rocks.com');
  });
